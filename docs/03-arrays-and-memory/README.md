# 03 – Arrays and Memory Layout

Arrays are the first data structure in Java and the foundation
for collections like ArrayList, HashMap, and queues.

This module explains:
- How arrays are stored in memory
- Primitive vs reference arrays
- Common bugs and interview traps
- How array behavior impacts performance and correctness

You should be able to:
✔ Explain how arrays are allocated in memory  
✔ Predict array mutation behavior  
✔ Avoid off-by-one and reference bugs  
✔ Answer OCP and interview questions confidently
